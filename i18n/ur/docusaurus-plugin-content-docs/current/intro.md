# خوش آمدید

**فزیکل اے آئی کتاب** میں خوش آمدید - روبوٹکس، ROS 2، اور اے آئی سے چلنے والے فزیکل سسٹمز سیکھنے کے لیے آپ کی جامع گائیڈ۔

## آپ کیا سیکھیں گے

| باب | موضوع |
|-----|-------|
| **باب 1** | فزیکل اے آئی اور ROS 2 کا تعارف |
| **باب 2** | Gazebo کے ساتھ سمیولیشن |
| **باب 3** | ویژن-لینگویج-ایکشن ماڈلز |

## شروع کریں

سائیڈبار سے کوئی باب منتخب کریں اور فزیکل اے آئی کی دنیا میں اپنا سفر شروع کریں۔
