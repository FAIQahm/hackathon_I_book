# باب 1: فزیکل اے آئی کا تعارف

فزیکل اے آئی کے بنیادی اصول سیکھیں۔

## اس باب میں

- فزیکل اے آئی کیا ہے؟
- روبوٹکس کی بنیادیں
- حقیقی دنیا کی ایپلیکیشنز

## فزیکل اے آئی کیا ہے؟

فزیکل اے آئی مصنوعی ذہانت کا وہ شعبہ ہے جو حقیقی دنیا میں روبوٹس اور مشینوں کو ذہین بناتا ہے۔
